convert example: pipe2sparky_2d
===============================

This example shows how to use nmrglue to convert a 2D NMRPipe file into a 
Sparky formatted file.

The data used in this example is available for 
`download. <http://nmrglue.googlecode.com/files/example_pipe2sparky_2d.zip>`_
