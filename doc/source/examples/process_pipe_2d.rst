process example: process_pipe_2d
================================

This example shows how nmrglue can be used to process NMR data.  In this 
script a 2D time domain NMRPipe file is processing into a 2D NMRPipe 
frequency domain file. 

The data used in this example is available for 
`download. <http://nmrglue.googlecode.com/files/example_process_pipe_2d.zip>`_
