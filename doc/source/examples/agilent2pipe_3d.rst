convert example: agilent2pipe_3d
================================

This example shows how to use nmrglue to convert a 3D Agilent/Varian data set
into a NMRPipe file.

The data used in this example is available for 
`download. <http://nmrglue.googlecode.com/files/example_agilent2pipe_3d.zip>`_
