interactive example: interactive_1d
===================================

This example shows how to use nmrglue and 
`matplotlib <http://matplotlib.sourceforge.net/index.html>`_ to create a 
simple interactive environment for examining a 1D spectrum from a NMRPipe 
file.

The data used in this example is available for 
`download. <http://nmrglue.googlecode.com/files/example_interactive_1d.zip>`_
